StudyBuddy - Documentation & Guide (Roman Urdu)

Yeh document aapko StudyBuddy project ka complete overview dega, taake aap as a beginner is code ko samajh sakein aur future mein modify kar sakein.

1. Firebase Setup Guide (Backend)
---------------------------------
Is project ko functional banane ke liye aapko Firebase connect karna hoga. 

Steps:
1. Google par "Firebase Console" search karein aur apne Gmail se login karein.
2. "Add Project" par click karein aur project ka naam "StudyBuddy" rakhein.
3. Project dashboard mein "Web" icon (</>) par click karke app register karein.
4. Wahan se "firebaseConfig" copy karke `js/firebase.js` mein paste kar dein.
5. "Authentication" mein ja kar "Email/Password" ko Enable karein.
6. "Firestore Database" mein ja kar "Create Database" karein aur "Test Mode" on kar dein.
7. (Optional) Agar PDF upload karna ho to "Storage" bhi on karna parega.

2. HTML & CSS Basics
---------------------
- HTML: Website ka dhancha (structure) banane ke liye use hota hai (Inputs, Buttons, Sections).
- CSS: Design aur khoobsurti ke liye. Humne `:root` mein colors define kiye hain taake ek hi jagah se poori website ka color badla ja sake.
- Responsive: Yeh website mobile aur desktop dono par sahi chalti hai (@media queries ki madad se).

3. JavaScript Logic
-------------------
- onAuthStateChanged: Check karta hai ke user login hai ya nahi.
- renderTasks() / renderNotes(): Database se data laa kar screen par dikhate hain.
- toggleModal(): Forms ko pop-up ki tarah kholne aur band karne ke liye.

5. Study Planner (H:M:S & Plans)
---------------------------------
- **Time Format:** Tasks mein Hours, Minutes aur Seconds teeno set kiye ja sakte hain.
- **Tabs (Daily/Weekly/Monthly):** Aap apne tasks ko date ke mutabiq filter kar sakte hain.
- **Symbols:** Time ke sath symbols (`h, m, s`) show hote hain.

6. Notes with PDF Support
--------------------------
- Aap notes mein Google Drive ka PDF link add kar sakte hain.
- "View PDF" button par click karne se PDF naye tab mein khul jati hai.

7. Smart Reminders & Sound
--------------------------
- Sham 4 PM ke baad system pending tasks ka alert deta hai.
- User ke ek click ke baad system "Buzz" sound play karta hai taake user ko yaad rahe.

8. SEO (Search Engine Optimization)
-----------------------------------
- Humne Meta Tags, `robots.txt` aur `sitemap.xml` add kiye hain taake Google aapki website ko asani se dhund sake.

9. Customization
----------------
- Colors badalne ke liye `css/style.css` mein `--primary-color` badlein.
- Admin badalne ke liye `js/auth.js` mein `ADMIN_EMAIL` badlein.
